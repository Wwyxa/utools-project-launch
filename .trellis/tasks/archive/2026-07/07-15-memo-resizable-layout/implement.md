## 验证命令

1. `npm run lint`
2. `npm run build`
3. 启动现有 Vite 开发服务并在桌面、窄视口检查布局与拖拽行为。

## 风险与回滚点

- 首个验证点：共享组合式函数接入后立即运行类型检查，确认事件与 Vue 生命周期类型正确。
- 第二验证点：待办覆盖按钮改造后构建并检查 hover、focus、编辑态是否互相遮挡。
- 回滚时可独立撤销 Memo 接入或共享组合式函数；本任务无数据迁移和存储副作用。
